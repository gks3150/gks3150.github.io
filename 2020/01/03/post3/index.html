JavaScript 기초 학습

자바스크립트는 HTML과 CSS를 통해 만들어진 웹 문서의 구조와 스타일에 변화를 주는 로직을 추가하는 기능을 수행하는데 사용되며 window object를 통해 각종 HTML element와 css에 접근하고 조작하는게 가능하다.
window object를 통해 어떻게 html에 접근하는 것이 가능한가? 이는 HTML이 DOM, Document Object Model 형식으로 작성되었기 때문에 가능하다.
DOM 이란? 컴퓨터가 문서를 잘 처리할 수 있도록 문서에 대한 구조를 약속해 놓은 것을 의미한다. DOM은 Tree형태로 구성되어 있는데 각각의 부모 노드(element)에 여러 개의 자식으로 구성되어 있다. 이는 HTML 문서를 작성할 때 태그가 중첩되는 것과 일맥상통하는 의미이다.

html

예를 들면, html 부모 태그에는 head, body인 자식 태그가 존재하고, body 태그에는 두 개의 div 자식 태그가 존재한다.
따라서 이러한 DOM 형태의 문서를 통해 구성된 Object를 탐색하는 것을 통해 자바스크립트를 활용할 수 있는 것이다.
